# sl1-id

## Usage

```
Usage: sl1-id [OPTION]... [USERS]
Print sl1 user information for the specified USERS,
or (when USER omitted) prints a list of all users.

  -v             version
  -h             display this and exit
```

## Example

```shell
sl1-id <user1> <user2>

# output:
sl1id=1(user1)
sl1id=6(user2)

```
